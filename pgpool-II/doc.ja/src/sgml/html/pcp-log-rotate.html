pcp_log_rotate

名前

pcp_log_rotate --  Pgpool-IIのログファイルをローテーションします

概要

pcp_log_rotate [options...]

説明

pcp_log_rotate Pgpool-IIのログファイルをローテーションします。

オプション

-s scope
--scope=scope

コマンドの与える影響範囲を指定します。

　　　　サポートされているコマンドスコープは以下です。 デフォルトは"local"です。

  • c, cluster : クラスタを構成するすべてのPgpool-IIのログファイルをローテーションします。

  • l, local : ローカルPgpool-IIのログファイルをローテーションします。

その他のオプション

pcp_common_options を参照してください。
